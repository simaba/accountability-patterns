# Example: Customer Support Agent

A customer support agent uses AI to draft responses and classify tickets.

## Accountability design
- Product lead accountable for release decision
- Operations lead accountable for runtime monitoring
- Support manager accountable for override on sensitive cases

## Override triggers
- legal or refund disputes
- abusive output
- repeated user dissatisfaction
- policy-sensitive cases

## Notes
This is lower risk than a safety-adjacent system, but still requires explicit ownership.
