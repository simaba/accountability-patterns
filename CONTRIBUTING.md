# Contributing

Thanks for your interest in improving this repository.

## Scope

This repository is intended as a practical pattern library for AI accountability design. Contributions should strengthen clarity, reuse, and operational usefulness.

## Good contributions

Examples of helpful contributions include:
- improving pattern clarity,
- adding well-scoped examples,
- correcting inaccuracies,
- improving diagrams or documentation structure,
- proposing new accountability design patterns with concrete operational value.

## Contribution expectations

Please keep contributions:
- concise,
- evidence-based,
- implementation-aware,
- free of proprietary or confidential material.

## Pull request guidance

When opening a pull request, please explain:
- what changed,
- why it improves the repository,
- and whether it changes the conceptual model, examples, or supporting assets.
